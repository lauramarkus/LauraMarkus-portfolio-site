my portfolio

Jubilee Austen Website

A one-page web design portfolio site to highlight a designer’s experience. It was built from scratch using only HTML and CSS based on a design comp. The site is divided into three simple sections with links to connect on social media and via email.

Rogue Pickings

A responsive multi-page website for a local farm-to-kitchen food truck coded using HTML and CSS with flexbox properties. Designs were extracted from a Photoshop design comp.

Unplugged

A three-page responsive site for a company that organizes corporate retreats in nature. Teams need time and a place to unplug and reconnect with what makes them tick. This site gives you the basics you need to know to get in touch and let them plan your next outing.
